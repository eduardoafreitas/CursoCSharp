# Curso C# Completo: Do Básico ao Avançado

Este repositório contém meus estudos, exercícios e projetos desenvolvidos durante o curso **C# Completo** da Cod3r. O objetivo é dominar a linguagem C# e o ecossistema .NET, explorando desde a lógica básica até recursos avançados e Orientaçăo a Objetos.



## Sobre o Curso
O curso aborda a linguagem C# de forma profunda, focando em boas práticas, algoritmos e no paradigma de Orientaçăo a Objetos (OO).

## Estrutura do Repositório
O aprendizado está dividido nas seguintes seçőes:

- [x] **01. Introduçăo:** Algoritmos, estrutura de dados e visăo geral do .NET.
- [x] **02. Configuraçăo do Ambiente:** Setup do Visual Studio e ambiente de desenvolvimento.
- [x] **03. Fundamentos:** Variáveis, tipos de dados, operadores e entrada/saída.
- [x] **04. Estruturas de Controle:** Estruturas condicionais (if/else, switch) e laços de repetiçăo (for, while).
- [x] **05. Classes & Métodos:** Início da jornada em OO (membros, construtores, `this`).
- [x] **06. Coleçőes:** Arrays, Listas, Filas, Pilhas e Dicionários.
- [x] **07. Orientaçăo a Objetos:** Pilares da OO (Encapsulamento, Herança, Abstraçăo e Polimorfismo).
- [x] **08. Métodos e Funçőes:** Lambdas, Delegates e programaçăo funcional em C#.
- [x] **09. Exceçőes:** Tratamento de erros com `try/catch`.
- [x] **10. Explorando API:** Manipulaçăo de arquivos (IO), DateTime e muito mais.
- [/] **11. Tópicos Avançados:** LINQ, Generics, Dynamics e Nullables.

> *Nota: O status `[x]` indica concluído e `[/]` indica em progresso.*

## Tecnologias Utilizadas
* **Linguagem:** C#
* **Plataforma:** .NET
* **IDE:** Visual Studio
