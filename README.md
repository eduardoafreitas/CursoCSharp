# Curso C# Completo: Do Básico ao Avançado

Este repositório contém meus estudos, exercícios e projetos desenvolvidos durante o curso **C# Completo** da Cod3r. O objetivo é dominar a linguagem C# e o ecossistema .NET, explorando desde a lógica básica até recursos avançados e Orientação a Objetos.



## Sobre o Curso
O curso aborda a linguagem C# de forma profunda, focando em boas práticas, algoritmos e no paradigma de Orientação a Objetos (OO).

## Estrutura do Repositório
O aprendizado está dividido nas seguintes seções:

- [x] **01. Introdução:** Algoritmos, estrutura de dados e visão geral do .NET.
- [x] **02. Configuração do Ambiente:** Setup do Visual Studio e ambiente de desenvolvimento.
- [x] **03. Fundamentos:** Variáveis, tipos de dados, operadores e entrada/saída.
- [x] **04. Estruturas de Controle:** Estruturas condicionais (if/else, switch) e laços de repetição (for, while).
- [x] **05. Classes & Métodos:** Início da jornada em OO (membros, construtores, `this`).
- [x] **06. Coleções:** Arrays, Listas, Filas, Pilhas e Dicionários.
- [/] **07. Orientação a Objetos:** Pilares da OO (Encapsulamento, Herança, Abstração e Polimorfismo).
- [ ] **08. Métodos e Funções:** Lambdas, Delegates e programação funcional em C#.
- [ ] **09. Exceções:** Tratamento de erros com `try/catch`.
- [ ] **10. Explorando API:** Manipulação de arquivos (IO), DateTime e muito mais.
- [ ] **11. Tópicos Avançados:** LINQ, Generics, Dynamics e Nullables.

> *Nota: O status `[x]` indica concluído e `[/]` indica em progresso.*

## Tecnologias Utilizadas
* **Linguagem:** C#
* **Plataforma:** .NET
* **IDE:** Visual Studio
